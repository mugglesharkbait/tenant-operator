## Key decisions

- **Module path:** `github.com/your-github-username/tenant-operator` — a deliberate, valid
  *placeholder* (angle brackets like `<github-username>` are not legal in Go import paths).
  It is just an identifier baked into `go.mod`; it does not require the GitHub repo to exist.
  Swap `your-github-username` for the real handle on publish. Full explanation of what the
  Go module/import path is and why it matters: see [SETUP.md](SETUP.md) Phase 3.
- **Local cluster:** minikube (not kind), since everything runs locally.
- **API:** group `platform`, domain `example.io`, version `v1alpha1`, kind `Tenant` → GVK `platform.example.io/v1alpha1/Tenant`. Cluster-scoped (it creates cluster-scoped namespaces).

## Code generation reference

We hand-write only two source files (`api/v1alpha1/tenant_types.go`,
`internal/controller/tenant_controller.go`). Everything below is generated by `controller-gen`
from the Go types + `// +kubebuilder:...` markers — do not hand-edit the outputs.

| Command | Tool invoked | Generates |
|---|---|---|
| `make generate` | `controller-gen object` | `api/v1alpha1/zz_generated.deepcopy.go` (Go deep-copy methods) |
| `make manifests` | `controller-gen crd rbac webhook` | `config/crd/bases/platform.example.io_tenants.yaml` (CRD) + `config/rbac/role.yaml` (operator ClusterRole) |
| `make install` | `kustomize` + `kubectl apply` | nothing — applies the CRD into the cluster |

Workflow rule: after editing the Go types or markers, run `make generate manifests`, then
re-`make install` if the CRD changed.

## Build complete

All 7 phases done. The operator turns a declarative `Tenant` into a provisioned, isolated
namespace (quota + default-deny network policy + scoped RBAC), self-heals managed-resource
drift, and cleans up via owner-reference garbage collection.

Possible next steps (from the README roadmap): finalizers, admission webhooks (reject a
Tenant with no owners — currently enforced only by CRD `minItems`), per-owner roles,
consume `displayName`, Prometheus metrics, package with Helm.
